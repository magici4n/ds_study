# 자료 구조

자료 구조 공부를 위해 만든 것으로
나에게 있어서 처음으로 공부 요약본을 올리는 것이기에 
많은 부족함이 있을 것이다.

언어는 파이썬을 사용하였으며
각 파일별(stack, 큐, Linked_list 등)에는 개념파일(readme),강의를 통한 일반적인 구조, 백준 예제 문제로 구성 되어있다.

개념은 한국외대 신찬수 교수님의 유투브 강좌를 참고하였다. 아래는 강의 링크이다.
